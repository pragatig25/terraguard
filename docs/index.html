Live mode — real pipeline data from this repository

Live mode shows real scan results generated by the TerraGuard CI pipeline. To populate it:

  1. Fork or clone this repo and add your ANTHROPIC_API_KEY secret in Settings → Secrets
  2. Create a branch and edit a .tf file in examples/terraform/ — e.g. open SSH to 0.0.0.0/0
  3. Open a pull request — the security-regression workflow scans, triages, and gates
  4. Merge to main — the baseline-capture workflow updates docs/data/dashboard.json
  5. Dashboard auto-deploys — switch to LIVE to see real posture data here

Only repo collaborators can trigger pipelines. Fork PRs from strangers are blocked.
